# MauiAppFit

Projeto desenvolvido em **.NET MAUI** para o registro e acompanhamento de atividades físicas, permitindo ao usuário armazenar informações sobre seus exercícios e evolução.

## Funcionalidades

* Cadastro de atividades físicas.
* Registro da data da atividade.
* Controle de peso corporal.
* Adição de observações sobre os treinos.
* Listagem das atividades cadastradas.
* Busca de atividades por descrição.
* Armazenamento local utilizando SQLite.

## Tecnologias Utilizadas

* C#
* .NET MAUI
* XAML
* SQLite

## Como Funciona

O aplicativo permite que o usuário cadastre informações sobre suas atividades físicas, incluindo descrição, data, peso e observações. Os dados são salvos em um banco de dados local SQLite e podem ser consultados posteriormente na tela de listagem, que também oferece uma ferramenta de busca para localizar registros específicos.

## Objetivo

Aplicar conceitos de desenvolvimento mobile com .NET MAUI utilizando o padrão MVVM, navegação entre páginas, persistência de dados com SQLite e atualização dinâmica da interface através de Data Binding.
